SDKMAN + direnv 自动切换 JDK 环境

标签： Configuration

在多项目并行开发的今天，A 项目用 Java 8，B 项目用 Java 17 已是常态。虽然 SDKMAN! 解决了”装”的问题，但”用”的过程依然不够优雅——每次进项目都要手动 sdk use，着实繁琐。

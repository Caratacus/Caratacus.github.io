👋 你好，我是 Caratacus

我是一名拥有十多年经验的后端开发工程师与架构师。在过去的职业生涯中，我曾涉足过新能源、教育、电力以及电商等多个行业领域。这些跨行业的业务场景积累，带给我丰富的系统设计与落地经验。

我的技术栈主要聚焦于 Java 生态，深度使用 Spring Boot 和 Spring Cloud 进行微服务架构的设计与实践。

曾作为公司的架构负责人，主导并亲手完成了核心业务系统从 0 到 1 的搭建过程。

在目前就职公司，主要负责核心系统的重构与架构升级。

🚀 关注领域

  • 架构与重构: 系统架构设计, 微服务架构, 系统深度重构
  • Java 生态: Spring Boot, Spring Cloud, JVM 调优, 高并发处理
  • 工程效能: 自动化构建, Git 工作流, 代码质量与规范
